# Mi primera web estática con Pelican

¡Hola! En esta práctica he aprendido a configurar un flujo de trabajo profesional para el despliegue de una **página web estática**.

## ¿Qué hemos hecho?
He configurado un entorno local con **Python** para utilizar **Pelican**, uno de los generadores de sitios estáticos (SSG) más potentes del mercado. El proceso ha incluido:

1.  **Inicialización**: Configuración del proyecto base.
2.  **Control de versiones**: Gestión del código mediante **Git** y **GitHub**.
3.  **Automatización**: Uso de **GitHub Actions** para que la web se compile y publique sola en cada `push`.

## Comparativa de tecnologías SSG
Durante el estudio de esta práctica, hemos explorado las tres tecnologías principales recomendadas:

* **Pelican**: Basado en **Python**, destaca por su sencillez y potencia para usuarios de este ecosistema.
* **Hugo**: Basado en **Go**, es conocido por su increíble velocidad de compilación.
* **Jekyll**: Basado en **Ruby**, es el estándar nativo de GitHub Pages.

> **Conclusión**: Una web estática es rápida, segura y económica, ideal para currículums o blogs técnicos.
